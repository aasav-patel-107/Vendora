# Vendora

Vendora is a demo web for role-based product management app built with Flask and PostgreSQL.  
Roles: Admin, Vendor, User

## Features
- Admin approval system
- Vendor product CRUD
- User login/register
- Responsive, modern UI

## Tech Stack
- Flask
- PostgreSQL
- SQLAlchemy
- Bootstrap 5

## Getting Started

```bash
git clone ...
pip install -r requirements.txt
python app.py
